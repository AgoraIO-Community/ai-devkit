# minimal-repo fixture

This is a small structural fixture showing the expected shape of:

- `AGENTS.md`
- `docs/ai/L0_repo_card.md`
- all 8 L1 files
- `docs/ai/L1/L2/_index.md`

Use it as a path and naming reference, not as a content-quality model.

The full ai-devkit repo is the richer live example. This fixture is intentionally
small so maintainers can test structure changes without reading the whole repo.
