# recipe-base fixture

This fixture shows the shape of a base recipe repo using the recipe profile.

It demonstrates:

- recipe fields in L0
- `docs/ai/RECIPE.md`
- the normal 8 L1 slots

Use it as a path and naming reference, not as a content-quality model.

Use it with [docs/standard/recipe-profile.md](../../docs/standard/recipe-profile.md).
